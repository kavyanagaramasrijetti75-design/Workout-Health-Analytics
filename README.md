# 🏋️ Workout & Health Analytics Dashboard

Power BI project that analyzes workout and health data through interactive dashboards to uncover fitness trends, performance insights, and health patterns.

---

## 📌 Project Overview
The Workout & Health Analytics project is a Power BI dashboard designed to analyze workout activities and health-related data. The project transforms raw fitness data into meaningful insights using interactive visualizations and analytics.

The dashboard helps users understand workout patterns, monitor performance metrics, and track overall health trends to support better fitness decisions.

---

## 🎯 Project Objectives
- Analyze workout activity and health data
- Track workout intensity and session duration
- Identify fitness trends and performance patterns
- Provide interactive and user-friendly dashboards
- Enable data-driven fitness improvement

---

## 📊 Dashboards Included

### 1️⃣ User Overview Dashboard
- Total users and activity summary  
- Health metrics overview  
- Key performance indicators (KPIs)

### 2️⃣ Workout Activity Dashboard
- Workout type distribution  
- Workout frequency analysis  
- Session duration insights  
- Workout intensity comparison  

### 3️⃣ Performance Metrics Dashboard
- Fitness performance trends  
- Scatter plot analysis  
- Gauge visualization for performance level  
- Advanced analytics insights
